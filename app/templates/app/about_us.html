About Us

Team Members Management is a simple Django project designed to help students learn CRUD operations, authentication, and clean UI patterns using Bootstrap.

What you can learn here

  • Create, edit, and delete team members with validation
  • Use Django authentication (login, logout, register)
  • Work with models and relationships (Departments and Members)
  • Build responsive layouts with Bootstrap 5
  • Add small UX touches like active links and dark mode

If you are following along as a student, explore the navigation links above and check the code comments in templates for explanations you can reuse in your own projects.
